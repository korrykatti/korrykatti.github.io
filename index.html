Hey there! You've stumbled upon a website made by yours truly, Korrykatti. Now, if you're expecting slick designs and CSS wizardry, you're in the wrong place, buddy.

See, CSS and I have an understanding: I don’t mess with it, and it doesn’t mess with me. So, I threw this site together without giving CSS a second thought. (Or even a first thought, really.)

What you see is what you get—no frills, no fuss, just pure, unfiltered HTML vibes. It’s like a sandwich with just the bread: not flashy, but it’ll fill you up... probably.

If you're still here, congratulations! You've either got too much free time or you're really intrigued by my lack of CSS skills. Either way, thanks for sticking around!

Maybe someday I’ll learn CSS and transform this site into something truly magnificent. Or maybe I'll just keep it like this as a monument to laziness. Who knows?
